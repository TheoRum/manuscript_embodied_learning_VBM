# README file for the following manuscript:

Title: Impact of technological immersion and sensorimotor engagement on performance and brain plasticity in short-term second language vocabulary training.

Journal: Neurobiology of Language

# Data
This repository contains the statistical maps from the FSL-VBM analysis between the following group comparisons reported in the manuscript:
- dVE versus iVR
- high-manipulator versus low-manipulator
as well as the GM masks for the whole brain and region-of-interest (ROI) analysis

Additionally, all the meta and behavioral data mentioned in the manuscript can be found in: meta_data.xlsx

A more detailed description of the whole analysis can be found in the manuscript.

# Software/Tools
The following software packages were used:
* FSL version v6.0.6
* N4 in Advanced Normalization Tools (ANTs) v2.4.2
* Python statsmodel v0.14.0
* Python Scipy v1.11.3
